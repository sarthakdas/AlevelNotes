Networks

The internet first started as a United States Defence Project called ARPANET to enable distant departments to communicate. The world wide web is a collection of web pages that reside on computers connected to the Internet.

Protocols and Standards

A protocol is a set of rules defining common methods data communication. These rules need to be standard in-order for them to communicate with each other. HTTPS has become the standard protocol for browsers. TCP/IP is also used worldwide and enables communication with any other computer connected to the internet regardless of location.

Structure of the Internet

The TCP/IP Protocol Stack

The Transmission Control Protocol / Internet Protocol stack is a set of networking protocols that work as 4 connected layers, passing incoming and outgoing data packets.

There are 4 layers:

  • Application Layer
  • Transport Layer
  • Network Layer
  • Link Layer

The Role of the Four Layers in the Stack

Various protocols operate at each layer of the stack, each with different roles. This is done so one layer can be changed and it will not effect the other layers. It is used to wrap send and unwrap packets.

Application Layer

This sits on top of the stack and uses protocols relating to the application being used to transmit data over a network. Common protocols in this layer are HTTPS, POP3, FTP.

Transport Layer

The transport layer uses TCP to establish an end-to-end connection with the recipient computer. The data is split into packets and labeled with:

  • A packet number
  • Total number of packets
  • Port number through which the packet should route through

If any packets are missing the transport layer can request a retransmission.

Network Layer

This is the IP layer this adds the source and destination IP addresses. Routers operate on to the network layer and will use these to forward the packet on to the destination. The IP to the port number forms a socket.
